Base for Django - MariaDB in docker

### Requirement

- docker
- docker-compose

### Usage

Build and run :

```
docker-compose up
```

Start creating django app :

```
docker-compose exec web python manage.py startapp myapp
```

and start coding :)

view your apps in port 8000
